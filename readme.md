# 🎬 MovieBot - Telegram бот для учёта просмотренных фильмов
**MovieBot** — это Telegram бот для совместного ведения базы данных просмотренных фильмов в небольших группах. Позволяет добавлять, удалять, искать и обновлять информацию о фильмах.

## ✨ Возможности

- ✅ **Добавление фильмов** с названием и датой просмотра
- 🔍 **Поиск** по названию или дате
- ✏️ **Обновление** информации о фильме
- 🗑️ **Удаление** записей из базы
- 📋 **Просмотр всех фильмов** в виде форматированной таблицы
- ⌨️ **Интерактивная клавиатура** для удобного управления
- 🗄️ **PostgreSQL** — надёжное хранение данных

## 📋 Требования

- Python 3.8 или выше
- PostgreSQL 13 или выше
- Аккаунт Telegram и токен бота от [@BotFather](https://t.me/botfather)

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/NikitaProkhvatilov/Movie_bot.git

cd Movie_bot
#### Перейдите в директорию Movie_bot
```
### 2. Настройка виртуального окружения
```
# Создание виртуального окружения
python -m venv venv

# Активация (Linux/macOS)
source venv/bin/activate

# Активация (Windows)
venv\Scripts\activate
```
### 3. Установка зависимостей

```
pip install --upgrade pip
pip install -r requirements.txt
```
### 4. Настройка конфигурации
Создайте файл `.env` в корне проекта:
```
# Telegram Bot Token (получить у @BotFather)
TOKEN=your_telegram_bot_token_here

# PostgreSQL настройки
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=your_password
DB_NAME=movie_bot_db
```
### 5. Применение миграций базы данных
```
alembic upgrade head
```
### 6. Запуск бота
```
python main.py
```
## 📖 Использование
### Начало работы
Запустите бота командой `python movie_bot.py`

В Telegram найдите вашего бота по имени

Отправьте команду `/start`

Используйте клавиатуру для выбора действий

### 🎯 Доступные команды

| Команда | Описание | Пример |
|:--------|:---------|:-------|
| 🚀 `/start` | Активация бота, показ клавиатуры | `/start` |
| ➕ `/добавить` | Добавить новый фильм | `/добавить "Интерстеллар" 5 февраля 2026` |
| ❌ `/удалить` | Удалить фильм из базы | `/удалить "Интерстеллар"` |
| ✏️ `/обновить` | Изменить информацию о фильме | `/обновить` → следуйте инструкциям |
| 🔍 `/поиск` | Найти фильм по названию или дате | `/поиск` → выберите тип поиска |
| 📋 `/фильмы` | Показать все фильмы в базе | `/фильмы` |

### 🔧 Технологический стек

#### Backend
- **Python 3.8+** — основной язык разработки
- **pyTelegramBotAPI 4.14+** — взаимодействие с Telegram API
- **PostgreSQL 13+** — реляционная база данных
- **SQLAlchemy 2.0.46** — библиотека для работы с базами данных (Core API)
- **Alembic** — инструмент для миграций базы данных
- **psycopg2-binary** — адаптер PostgreSQL для Python

#### Утилиты
- **python-dotenv** — управление переменными окружения
- **tabulate** — форматирование таблиц для вывода
- **logging** — встроенное логирование событий
